# Exercice 1 - Gestionnaire de Contacts

## Description
Ce programme permet de gérer une liste de contacts.
Chaque contact possède un nom et un numéro de téléphone.

## Fonctionnalités
- Ajouter un contact (nom + numéro de téléphone)
- Rechercher un contact par son nom
- Afficher le numéro de téléphone du contact trouvé

## Fichiers
- `Contact.java` : la classe Contact (nom et numéro de téléphone)
- `Main.java` : contient la classe ContactsManager et le point de départ du programme

## Comment lancer le programme
1. Ouvrir le projet dans IntelliJ
2. Lancer le fichier `Main.java`
3. Le programme affiche automatiquement le numéro du contact recherché

## Langage utilisé
Java
